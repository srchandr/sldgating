# light-transport-matrix-pbrtv3

## How to use

1. install docker and docker-compose

1. Clone this repository
   ``` bash
   clone https://github.com/qlab2020/light-transport-matrix-pbrtv3.git
   ```
1. Build an image
   ```bash
   docker-compose build
   ```
1. Run
   ```bash
   docker-compose run
   ```
